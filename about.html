About Tagtaly

News that matters. Data that shows it.

What Is Tagtaly?

Tagtaly is an automated news analysis system that transforms the chaos of daily UK news into clear, shareable data visualizations.

Every day, we:

  • Collect 500+ articles from major UK news sources
  • Analyze topics, sentiment, and emerging trends
  • Detect the most dramatic story angles using proprietary algorithms
  • Visualize the results in 3 Instagram-ready charts
  • Deliver them at 7 AM UTC, ready to post

How It Works

1. Data Collection

Sources: BBC, Guardian, Sky News, Independent

Volume: 200-500 articles daily

Method: RSS feed aggregation with deduplication

2. Analysis

Topic Classification: 9 categories (Politics, Health, Economy, Crime, Energy, etc.)

Sentiment Analysis: Positive, neutral, or negative using TextBlob NLP

Trend Detection: Week-over-week comparisons and pattern recognition

3. Story Detection

We run 5 proprietary algorithms to find the most dramatic angles:

  • Surge Detection: Topics exploding week-over-week
  • Political Scorecard: Who's dominating headlines
  • Record Alerts: Highest/lowest ever mentions
  • Sentiment Shifts: Mood changes by topic
  • Media Bias: What each outlet covers

4. Visualization

Format: Mobile-first Instagram portrait (1080x1350px)

Style: Bold, minimal, data-focused

Output: 3 charts daily + social media captions
